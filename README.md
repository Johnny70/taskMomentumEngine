
# Personal Workflow OS (PWOS)

En modulär, AI-assisterad produktivitetsplattform byggd med SvelteKit.

## Funktioner

- Dashboard med widgets och länkar
- Task System (skapa, lista, ta bort tasks)
- Project System (skapa, lista, ta bort projekt)
- Focus Areas (skapa, lista, ta bort fokusområden)
- Timer System (starta/stoppa timer på tasks)
- Momentum Engine (beräkning och visning av momentum)
- Deep Work Engine (analys av långa, fokuserade sessioner)
- Routing till separata modulsidor

## Kommande moduler

- Totem System (visuell representation av framsteg)
- AI-integration (prediktioner, förslag, analys)
- Offline-läge
- Energy Module (valbar)
- Onboarding
- Snapshots
- Mobilanpassning

## Utveckling

Installera beroenden:

```sh
npm install
```

Starta utvecklingsserver:

```sh
npm run dev
```

Bygg för produktion:

```sh
npm run build
```

Förhandsgranska produktion:

```sh
npm run preview
```
